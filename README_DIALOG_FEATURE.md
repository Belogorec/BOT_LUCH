## 🔧 Валидация

### Телефон
- Форматы: +7XXXXXXXXXX, 8XXXXXXXXXX, 7XXXXXXXXXX
- Автоматическая нормализация в E.164 (+7XXXXXXXXXX)
- Проверка что это русский номер

### Дата
- Форматы: DD.MM, DD.MM.YYYY, YYYY-MM-DD
- Автоматическое добавление года (текущий или следующий)
- Не позволяет бронировать в прошлое
- Максимум 365 дней в будущее

### Время
- Формат: ЧЧ:МММ
- Проверка диапазона работы (примерно 11:00 - 23:00)

### Количество гостей
- От 1 до 20 человек
- Только целые числа

## 📊 База данных

### Используемые таблицы:
- `bookings` — новые брони сохраняются сюда
- `guests` — гость автоматически добавляется/обновляется
- `booking_events` — события логируются
- `pending_replies` — состояние диалога сохраняется

### Новые события в `booking_events`:
- `CREATED_FROM_BOT` — бронирование создано из диалога
- `ADMIN_NOTIFIED` — админу отправлено уведомление
- `ADMIN_NOTIFY_FAILED` — ошибка отправки админу
