# 📱 Обновление: Контакты + Меню кнопки

## ✅ Что добавлено

### 1. Запрос контактов через встроенную кнопку Telegram
- Пользователь нажимает `/start` или меню кнопку "Забронировать стол"
- Вместо ввода вручную видит встроенную кнопку Telegram: **"📱 Поделиться контактом"**
- После поделки контактом автоматически переходит на запрос даты
- Резервный вариант: можно ввести имя и телефон вручную

### 2. Инлайн кнопки главного меню
При `/start` пользователь видит:
- 🍸 **Забронировать стол** - запускает диалог бронирования
- 📅 **Ближайшие события** - заглушка (позже добавить информацию)
- 📋 **Меню ресторана** - заглушка (позже добавить PDF)

### 3. Улучшена логика сохранения promo параметра
- Promo ссылки сохраняются и не перезаписываются
- Прежние promo активируются даже при повторном /start

**РАБОТАЕМ ТОЛЬКО С ЭТИМИ ФАЙЛАМИ (без дублей):**

## 📝 Как это работает

### Сценарий 1: Пользователь нажимает /start
```
Пользователь: /start
    ↓
Бот: "👋 Добро пожаловать в LUCHBAR!"
     [Кнопки: Забронировать | События | Меню]
```

### Сценарий 2: Пользователь нажимает "Забронировать стол"
```
Пользователь нажимает кнопку
    ↓
Бот: "Пожалуйста, поделитесь контактом"
     [Встроенная кнопка: "📱 Поделиться контактом"]
     
Пользователь нажимает кнопку поделка контаком
    ↓
Бот: "Когда вы хотите забронировать стол?"
     (автоматически перешел на запрос даты)
```

### Сценарий 3: Пользователь вводит текст вместо контакта
```
Пользователь: "Иван"
    ↓
Бот: "Пожалуйста используйте кнопку контакта."
     "Или введите свое имя..."
     
Пользователь: "Иван"
    ↓
Бот: "Теперь введите телефон..."
```
